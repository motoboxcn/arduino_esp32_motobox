# 开发相关文档

本目录包含ESP32-S3 MotoBox轻量化版本的开发、调试和系统说明相关文档。

## 调试系统
- `Debug_Output_Optimization.md` - 调试输出优化方案
- `调试输出控制.md` - 调试输出控制说明

## 系统说明
- `数据采集系统使用说明.md` - 数据采集系统完整使用说明
- `轻量化版本改造总结.md` - 轻量化版本改造总结

## 开发特性
- **串口调试**：支持多级别调试输出
- **性能监控**：CPU、内存使用率监控
- **数据采集**：传感器数据采集和处理
- **系统诊断**：设备状态诊断和故障排查

## 调试级别
| 级别 | 说明 | 用途 |
|------|------|------|
| ERROR | 错误信息 | 系统错误和异常 |
| WARN | 警告信息 | 潜在问题提示 |
| INFO | 一般信息 | 系统状态和事件 |
| DEBUG | 调试信息 | 详细调试数据 |
| VERBOSE | 详细信息 | 完整执行流程 |

## 数据采集系统
- **实时采集**：传感器数据实时采集
- **数据处理**：滤波、校准、融合算法
- **存储管理**：本地存储和云端同步
- **性能优化**：低功耗和高效率设计

## 轻量化改造
### ✅ 保留功能
- Air780EG 4G通信
- IMU传感器和GNSS定位
- SD卡存储和OTA升级
- 电源管理和LED指示
- 音频提示和罗盘功能

### ❌ 移除功能
- TFT显示屏
- BLE通信
- WiFi配置
- 按钮控制
- UI界面库
