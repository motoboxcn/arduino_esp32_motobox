# 音频系统文档

本目录包含ESP32-S3 MotoBox轻量化版本的音频提示系统相关文档。

## 语音功能
- `欢迎语音包使用说明.md` - 欢迎语音包使用指南
- `新语音功能使用说明.md` - 新版语音功能说明
- `音频功能使用说明.md` - 音频功能完整说明

## 开发工具
- `语音文件制作指南.md` - 语音文件制作和转换指南

## 核心功能
- **语音反馈**：设备状态语音提示
- **操作确认**：关键操作语音确认
- **告警提示**：异常状态语音告警

## 音频特性
- **低功耗播放**：优化音频播放功耗
- **多语言支持**：支持中英文语音包
- **音量控制**：可调节音量大小
- **静音模式**：支持静音操作

## 语音提示场景
- 设备启动/关闭
- 网络连接状态
- 电量低电量告警
- GPS定位状态
- 系统异常提示

## 文件格式
- 支持WAV格式音频文件
- 采样率：16kHz
- 位深度：16bit
- 声道：单声道
