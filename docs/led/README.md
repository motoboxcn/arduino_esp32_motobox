# LED调试系统文档

本目录包含ESP32-S3 MotoBox轻量化版本的LED状态指示系统相关文档。

## LED调试系统
- `led_debug_system.md` - LED调试系统完整说明
- `led_debug_quick_start.md` - LED调试快速入门
- `led_charging_status.md` - LED充电状态指示

## 资源文件
- `led.png` - LED状态指示图解

## 核心功能
- **状态指示**：设备运行状态可视化
- **调试辅助**：开发调试状态显示
- **充电指示**：电池充电状态显示
- **网络状态**：4G连接状态指示

## LED状态定义
| 状态 | 颜色 | 闪烁模式 | 说明 |
|------|------|----------|------|
| 启动中 | 蓝色 | 快闪 | 系统初始化 |
| 正常运行 | 绿色 | 慢闪 | 设备正常工作 |
| 网络连接 | 蓝色 | 常亮 | 4G网络已连接 |
| GPS定位 | 黄色 | 慢闪 | GPS定位中 |
| 充电中 | 红色 | 呼吸灯 | 电池充电 |
| 低电量 | 红色 | 快闪 | 电量不足 |
| 系统错误 | 红色 | 常亮 | 系统异常 |

## 硬件特性
- **PWM控制**：支持亮度调节
- **RGB LED**：多色状态指示
- **低功耗设计**：休眠时自动关闭
- **可配置**：支持自定义状态模式
