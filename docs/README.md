## 🎯 轻量化版本特点

### ✅ 核心功能
- **4G通信**：Air780EG模块，MQTT数据上报
- **智能定位**：IMU + GNSS融合定位，轨迹记录
- **电源管理**：双电源模式，智能休眠唤醒
- **存储系统**：SD卡数据记录，OTA固件升级
- **状态指示**：LED调试系统，音频语音提示
- **罗盘功能**：方向检测，导航辅助

### ❌ 移除功能
- TFT显示屏、BLE通信、WiFi配置
- 按钮控制、UI界面库、TPMS支持

## 🔋 电源特性
- **工作电压**：3.3V - 4.2V
- **待机功耗**：< 50μA
- **工作功耗**：< 100mA
- **电池续航**：7-30天（取决于使用频率）
